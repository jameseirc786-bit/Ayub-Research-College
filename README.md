This is a professional, full-stack web application developed for Govt College for Women Ayub Research, Faisalabad. The platform is designed to streamline the student admission inquiry process and provide an official digital presence for the institution.

Features
Responsive Frontend: A modern, mobile-friendly design built with Bootstrap 5.

Admission Portal: An active inquiry form where students can submit their details.

Admin Dashboard: A secure area for staff to view and manage student applications in real-time.

Dynamic Database: Uses SQLite to store and retrieve student records efficiently.

Professional Aesthetics: Includes inspirational academic quotes and high-quality imagery.

Tech Stack
Backend: Python (Flask)

Database: SQLAlchemy (SQLite)

Frontend: HTML5, CSS3, Bootstrap 5

Icons/Fonts: Google Fonts

How to Run Locally
Clone the repository:
git clone 

Install required libraries:
pip install flask flask-sqlalchemy

Run the application:
python app.py
